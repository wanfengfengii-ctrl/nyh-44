## 1. 产品概述
本项目是一个沿海灯塔/雾号钟声传播模拟系统，用于可视化模拟不同天气条件下钟声在海洋环境中的传播路径、可达范围和信号衰减。
- 主要目的：为海事科研人员、港口管理人员提供直观的声学传播可视化工具，辅助评估灯塔/雾号的覆盖效果
- 核心价值：通过交互式布局和实时计算，帮助用户理解风向、风速、湿度、频率等参数对钟声传播的影响

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，直接使用 | 布置点位、调整参数、模拟传播、保存/加载方案 |

### 2.2 功能模块
1. **布局编辑器（主视图）**：Cytoscape.js 驱动的交互式地图，支持放置海岸点位、岩壁、港口、船只
2. **参数控制面板**：天气参数输入（风向、风速、湿度、鸣钟频率）
3. **传播可视化层**：传播扇区覆盖图、衰减强度热力显示
4. **数据分析图表**：Chart.js 展示不同方向的信号强度极坐标图
5. **方案管理**：保存/加载完整方案（含点位布局和天气参数）

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页（模拟工作台） | 顶部导航栏 | 项目标题、保存/加载按钮、帮助入口 |
| 主页（模拟工作台） | 布局画布 | Cytoscape.js 交互式画布，支持拖拽放置/移动元素 |
| 主页（模拟工作台） | 左侧工具栏 | 元素类型选择（灯塔/点位/岩壁/港口/船只）、删除工具 |
| 主页（模拟工作台） | 右侧参数面板 | 风向滑块/旋钮、风速输入、湿度滑块、鸣钟频率输入 |
| 主页（模拟工作台） | 传播覆盖层 | 在画布上实时渲染传播扇区、衰减区域、遮蔽区域 |
| 主页（模拟工作台） | 底部图表区 | Chart.js 极坐标图展示各方向信号强度 |
| 主页（模拟工作台） | 验证提示区 | 实时显示输入错误和冲突警告 |

## 3. 核心流程
用户进入主页后，从左侧工具栏选择元素类型，在画布上点击放置灯塔、点位、岩壁、港口和船只。随后在右侧面板调整天气参数（风向、风速、湿度、频率），系统实时计算并渲染钟声传播扇区和强度分布。用户可保存当前方案供后续加载使用。

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深海蓝 (#0A2463) 作为主色，金色 (#FFD700) 点缀，搭配海雾灰 (#8D99AE) 中性色
- 按钮风格：圆角矩形按钮，悬停时有发光效果（模拟灯塔光晕）
- 字体：标题使用 Playfair Display，正文使用 IBM Plex Sans，体现专业海事风格
- 布局风格：三栏布局（左工具栏 + 中央画布 + 右参数面板），底部悬浮图表区
- 图标风格：使用 lucide 图标，结合海事符号（锚、灯塔、波浪等）

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 主页 | 顶部导航栏 | 深色背景，金色品牌 Logo，右侧保存/加载按钮组 |
| 主页 | 左侧工具栏 | 卡片式按钮列表，选中态高亮发光，支持拖拽 |
| 主页 | 中央画布 | 深蓝渐变背景（模拟海洋），Cytoscape.js 交互层，传播扇区半透明叠加 |
| 主页 | 右侧参数面板 | 白色卡片，分组表单，滑块+数值输入联动，旋钮式风向选择 |
| 主页 | 底部图表区 | 半透明深色卡片，极坐标雷达图，动画过渡 |

### 4.3 响应式设计
- 桌面端（1280px+）：三栏完整布局，所有模块同时可见
- 平板端（768-1279px）：工具栏和参数面板折叠为可展开侧边栏
- 移动端（<768px）：单栏垂直布局，画布占满主要空间，参数面板通过底部抽屉打开

### 4.4 动画与交互细节
- 元素放置：缩放出现动画 + 落点涟漪效果
- 风向变化：传播扇区平滑旋转变换（CSS transition）
- 参数滑块：实时数值反馈气泡
- 信号强度：颜色渐变脉冲动画（强信号快闪、弱信号慢闪）
